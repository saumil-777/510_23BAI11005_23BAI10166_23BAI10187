# 🚀 Hybrid Approach to Warehouse Robot Path Planning  
### A Comparative Study of A* Algorithm and Reinforcement Learning (Q-Learning)

---

## 📌 Project Overview
Efficient path planning is a critical component in modern warehouse automation systems, where robots are required to navigate complex environments with obstacles while optimizing time and distance.

This project presents a **hybrid path planning framework** that combines:
- **A\*** (a classical heuristic-based algorithm)  
- **Q-Learning** (a reinforcement learning technique)

The objective is to analyze, compare, and integrate both approaches to determine the most efficient navigation strategy for a warehouse robot.

---

## 🎯 Objectives
- Design a **grid-based warehouse simulation environment**
- Implement and evaluate **A\*** for optimal pathfinding  
- Implement **Q-Learning** to enable adaptive decision-making  
- Perform a **comparative analysis** based on performance metrics  
- Develop a **hybrid selection mechanism** for improved efficiency  

---

## ⚙️ Methodology

### 🔹 A* Algorithm (Deterministic Approach)
- Uses heuristic (Manhattan distance) for efficient search  
- Guarantees the **shortest path** in static environments  
- Fast and reliable for known maps  

---

### 🔹 Q-Learning (Reinforcement Learning Approach)
- Learns optimal actions through **trial-and-error interactions**  
- Uses:
  - Rewards (goal reached)  
  - Penalties (invalid moves, longer paths)  
- Capable of adapting to **unknown or dynamic environments**  

---

## 🔗 Hybrid Strategy
The hybrid approach integrates both methods by:
- Executing both A* and Q-Learning  
- Comparing their performance (path length & efficiency)  
- Selecting the **most optimal path dynamically**

👉 This ensures:
- Reliability (from A*)  
- Adaptability (from RL)  

---

## 🏗️ Environment Design
- 2D grid representing a warehouse layout  
- Obstacles simulate **shelves and storage racks**  
- Defined:
  - Start position  
  - Goal position  
- Movement allowed in four directions (up, down, left, right)

---

## 📊 Performance Metrics
The project evaluates both approaches based on:
- ✅ Path Length  
- ⏱ Execution Time  
- 🔄 Learning Efficiency (for Q-Learning)  
- 📍 Navigation Accuracy  

---

## 🖥️ Output Features
The system provides:
- Path generated by A*  
- Path generated by Q-Learning  
- Comparative analysis of both methods  
- Execution time measurement  
- Hybrid decision output  
- Grid-based visualization of paths
